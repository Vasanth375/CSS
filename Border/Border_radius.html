This is an example of the border property Border-radius is the property used to make our border curvy shape on top-left, top-right, bottom-left, bottom-right

Here - border-radius property has two values 10px and 10%
10px is for top-left and bottom-right
10% is for top-right and bottom-left

Border-radius most commonly used to convert boxes into circles by setting border-radius property to it's half of the length of a square element

border-style also have the attributes none and hidden both have the same effect except hidden works for border conflict resolution for table elements. In a table element with multiple borders, none has the lowest priority (meaning in a conflict, border would show) and hidden has the highest priority (meaning in a conflict, border would not show).
